На нативном JavaScript

07.06.2019

Начал изучать нативный javascript. Достаточно интересный язык, хоть иногда у меня случался когнитивный диссонанс, так как все же упор я делаю на php, но если откинуть эти мелочи – языки достаточно похожи.

Эта небольшая мини-игра фактически это мое первое приложение написанное на js в рамках его изучения. Исходные данные следующие:

  • Возможно задавать время (по умолчанию выставленно 5 секунд).
  • За указанный промежуток времени необходимо кликнуть по максимальному количеству генерируемых квадратов.
  • В процессе игры необходимо заблокировать input, для того что не было возможность сбить запущенный таймер.
  • Генерируемые div должны быть разного размера, цвета и появляться в заданной области.
Репозиторий с исходиками игры, если лень ковыряться в исходном коде этой страницы: «Click me»

Сам скрипт отвечающий за функционал игры достаточно прост, хотя при его написание мне так не казалось :). Если разбить на последовательность действий, то это:

  1. Забираю необходимые нам DOM элементы через querySelector('...');.
  2. После чего вешаю прослушку событий addEventListener ( два 'click' и одно 'input').
  3. И описываю функционал мини-игры.

Если отбросить мелкие функции – скрыть/показать кнопку запуска, рандома ( для генерации элементов различного цвета и размера) и двух небольших функций, которые работают с заголовком (скрывают выставленое время, а после окончания возвращают в него полученный результа получилось 3 основные функции.

function startGame() {...} – в который указаны параметры для начала игры: задано начальное значение для счетчика, скрытие кнопки, блокировка input и старт таймера.

function renderBox() {...} – отвечает за генерацию элементов в нужной области. В функции указываем тип создаваемого элемента (div), с помощью function getRandom() {...} реализованно случайная генерация размера блока, а так же место его появления. Сначала цвет я хотел реализовать через рандом: let randomColor = getRandom(100000, 999999); box.style.background = '#' + randomColor; но в последствие отказался, создав массив с красивыми линейными градиентами.

function endGame() {...} – в ней до безобразия все просто. Останавливаем таймер, передаем результат игры в заголовок, разблокируем input.

И как я мог не дать сыграть в созданную игрулю ;).

Время игры: 5.0 с.

Ваш результат:

Так как это моя первая попытка реализоваться приложение на нативном js я не стал навешивать на неё ещё больше функционала. В дальнейшем я хочу эту игру проапгрейдить :) и привязать к ней таблицу, в которой будет указанно: имя - результат.

Для этого уже потребуется не только javascript, но и php, mysql. А так как они не доступны на github pages модернизацию этой игры буду делать у себя на хостинге, а ссылку обязательно размещу здесь.
