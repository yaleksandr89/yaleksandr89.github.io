Веб разработка

Веб-разработкой стал заниматься относительно недавно, меньше года назад. Изначально это было больше хобби, чем цель стать back-end специалистом

Написать скрипт, сделать шаблон для WordPress, сверстать html шаблон по psd макету.

Но дальше больше, основную часть своих усилий я переложил на изучение back-end (PHP, MySql, JS) и через некоторое время понял, что без «базы» front-end будет будет тяжело.

Поэтому не так давно восстановил пробелы в HTML5 и CSS3 и изучил CSS framework: Bootstrap 4.3. На текущий момент неплохо знаю back-end составляющую WordPress на фрилансе неоднократно делал шаблоны страниц, так же выполнял небольшие задачи, в основном, по написанию скриптов или добавлению функционала в WP. Имею практический опыт работы с:

  • PHP
  • MySql
  • JavaScript
